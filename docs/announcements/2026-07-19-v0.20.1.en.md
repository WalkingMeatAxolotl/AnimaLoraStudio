Fixes NaViT packed training failing to start in v0.20.0

### Fixed

- **NaViT packed training failed to start (#450)**
  In v0.20.0, training runs with NaViT packing (`navit_packing`) enabled crashed on the first step with `NameError: name 't5_attn' is not defined`. This is now fixed: no config changes are needed, and previously built latent caches are reused as-is.
