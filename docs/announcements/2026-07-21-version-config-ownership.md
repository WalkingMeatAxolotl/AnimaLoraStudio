title: 训练版本的配置不再自动跟随全局设置
pin: true
version: "0.21.0"
---
从 v0.21.0 起，训练版本保存过的配置不再被系统改写——包括点「开始训练」入队的那一刻。**大多数人什么都不用做**：没在全局设置里换过模型、也没用过训练配置里「触发词」字段的话，这篇可以直接跳过。

**如果你在全局设置里换了底模 / 文本编码器 / VAE**

以前已有的训练版本会跟着一起变，现在不会了。去版本的训练配置页，路径字段的值与全局当前值不一致时，标签右侧会出现「恢复默认」链接，点一下对齐；也可以用字段旁的「选择模型」从已下载的模型里直接选，或手填路径。

**如果你依赖训练配置里的「触发词」字段自动加进采样图**

该字段已退役，采样 prompt 不再自动带上触发词。把触发词写进采样 prompt 的开头即可。打标页的触发词（prepend 进每张 caption）不受影响。
