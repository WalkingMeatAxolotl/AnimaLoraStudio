Fixes an error when uploading an image zip from the browser on Python 3.10

### Fixed

- **Fixed an error uploading an image zip from the browser on Python 3.10, which prevented images from being imported (#284)**
  - On Python 3.10, uploading a zip of images from the dataset page failed with an error and the images couldn't be imported (uploading single images was unaffected). This is now fixed. Python 3.11 and later were never affected.
