应用内公告栏 · 访问地址迁回根路径 · 设置页即时生效 · 测试页加载提速 · LoRA 评估（实验性）· 训练与界面多项修复

### 新增

- **应用内公告栏：右上角铃铛集中看更新与迁移提示（#336, #337, #339）**
  此前版本更新了什么、撞到行为变化都没有主动提示；现在每次更新会自动弹一次公告，铃铛也可随时打开。

  - 左侧版本列表 + 右侧正文，正文支持完整 markdown（标题 / 列表 / 链接 / 代码）。
  - 更新日志从原来的「去设置里翻」统一进公告栏，并补齐了历史版本的英文（英文用户翻旧版本不再回退中文）。
  - 公告分「更新日志 / 公告 / 迁移」三类；需要你注意或操作的行为变化（如本版的访问地址迁移）会单独置顶。
- **LoRA 训练评估（实验性）：训练后自动给 checkpoint 算质量指标（#138, #319, #322, #323, #324）**
  作为后续评估相关功能的基础先落地。当前指标对动漫域的判别力还有限、仅供参考，默认按需开启，不是本版重点。

  - 训练后自动出评估样图并算 CLIP-T / CLIP-I / DINO-I，另加动漫域的 tag 召回率与角色一致性（CCIP），并与纯底模 baseline 对照看净增益。
  - 可在项目里手动圈定验证集；评估进度 / 日志接入统一的任务日志抽屉。
- **创建项目可自定义 slug + 跨页保留已选中项目（#333）**
  全中文项目名以前落盘会塌成 `project`、项目一多不好找；选好项目切到测试 / 队列又会丢失选中。

  - 新建项目弹窗加「项目 slug」输入框，实时校验并预览最终落盘名（留空仍按标题自动派生）。
  - 选中的项目切到队列 / 测试等全局页时保留在侧边栏，不用回列表重选。
- **大图预览底部显示「第几张 / 共几张」（#326）**
  下载 / 预处理 / curation / 正则集 / 训练采样 / XY 网格等处看大图、用 ←/→ 翻页时，底部新增当前位置计数。

### 变更

- **访问地址迁回根路径，`/studio/` 前缀移除（#334）**
  打开地址从 `localhost:8765/studio/` 变为 `localhost:8765/`，顺带修好「把容器端口反代到根」这类部署（如 ModelScope / HuggingFace 创空间）打不开的问题。

  - 旧 `/studio/...` 链接本版仍会自动跳转到新地址，下个版本移除，建议更新书签。
  - 详见置顶的「访问地址改为根路径」迁移公告。

### 改进

- **全局设置页改为即时生效，去掉底部保存条（#340）**
  开关 / 下拉改了即时保存，文本框 / 数字框失焦或回车时保存，右下角有保存状态指示，不再需要点「保存」。

  - 删 preset、恢复词典默认、移除自定义本地模型等破坏性操作会先弹确认。
- **测试页打开明显提速（#327, #328）**
  历史图多、项目 / 版本多时打开测试页卡顿的两处根因已修。

  - 历史栏改为虚拟滚动，元数据直接读 PNG 文本块（本机实测历史扫描 15.4s → 0.12s）。
  - LoRA 列表改为按需懒加载，不点开 LoRA 选择器就不再发那一批项目 / 版本请求。
  - XY 出图开始后结果定格，中途改坐标轴只影响下次生成，不再串改已生成的结果。

### 修复

- **修复 V100 等 fp16 显卡的训练精度映射 + GradScaler（#335）**
  修正 fp16 路径的 dtype 映射与 GradScaler 持久化，让仅支持 fp16 的显卡（如 V100）能正常训练。
- **训练缓存阶段的 VAE 分块日志不再刷屏 / 误导（#332）**
  做 latent 缓存时每张图都打一条分块日志（200 张图刷 200 条）、且写着「已用 > 总显存」像是显存不足。分块本身是对的（在规避显存换页崖），本版把这类日志去重并降级，不再刷屏和误导。
- **补正则 AI 先验取消按钮 + 设置抽屉点侧栏收起 + 版本面板加载防跳（#329）**

  - 正则集「AI 先验生成」运行中补上取消按钮（此前只有 booru 抓取有）。
  - 设置抽屉点左侧侧边栏也能收起（此前蒙版只盖主内容区）。
  - 版本「更新内容」面板加载 / 切换版本时高度不再跳动。
