预处理涂抹 / Mask 笔刷 · 单图查看缩放平移 · bundle 可打包 latent 缓存 · 版本切换一致性修复 · Ctrl+C 干净退出

### 新增

- **预处理「涂抹」工具：取色笔刷覆盖文字 / 水印（#393）**
  预处理页新增第五个工具「涂抹」：用取色笔刷把图上的文字、水印、杂物实心涂掉，避免 LoRA 学入。支持 Alt 点击吸管取色、软边笔刷、逐图撤销重做（Ctrl+Z）；改动先留在内存，可逐图或一键保存全部。
- **Mask 笔刷：标记不想让 LoRA 学习的区域（#394, #400）**
  涂抹页新增 Mask 模式：用笔刷把易崩坏的手、无关背景等区域标记出来，配橡皮擦与覆盖率提示（超过 50% 会警示）。mask 以 `{图名}.mask` 文件与图片同目录保存，自动跟随裁剪、放大等预处理操作（还原原图时删除）；bundle 导出可选一并打包。

  - 本版训练还不使用 mask（按 mask 加权 loss 的训练支持在后续版本推出），现在画好的 mask 会完整保留待用。
- **导出 bundle 可打包 VAE latent 缓存（#391）**
  导出 bundle.zip 时，训练集 / 正则集可分别勾选把 VAE latent 缓存（`.npz`）一并打包；导入到另一台机器后直接复用、跳过重新 encode（encode 慢的机器收益最大）。包体积会明显变大（1024² 每张约 1 MB，开翻转增广再翻倍）。
- **打标页显示验证集打标状态（#392）**
  版本有验证集图片时，打标页右栏新增「验证集打标状态」：打标进度与「此轮需要打标」按数据集分开统计；打标范围选「验证集」时训练集侧不再显示「—」。

### 改进

- **单图查看全面获得缩放平移（#396）**
  全站单图查看（生成页全屏查看、各页画廊预览放大、打标页内嵌大图）统一支持滚轮缩放（以指针为中心）、拖拽平移、双击在适配 / 100% 间切换；打标页大图升级为原图分辨率，放大看局部不再糊。

  - 查看器内点击图片不再关闭弹层（点击与拖拽平移无法两全）；关闭用 × 按钮、Esc 或点击边缘遮罩。
- **tag 自动补全：候选只在输入时弹出 + 全局开关（#401）**
  点击输入框挪光标不再弹出候选挡住正在读的内容，只有输入变化后才弹；候选已弹出时点击输入框会立即收起。设置 → Tag 翻译词典新增「输入时自动补全 tag」开关，可全站关闭。
- **评估失败可从日志条直接重试（#397）**
  训练后评估失败时，评估日志条上现在有「重试」按钮，一键重跑本轮全部评估；此前只能手动重新选 checkpoint 跑一遍。任务详情头部的「重试」对训练任务改名「重新训练」，明确它是用相同配置从头训练、而不是重跑评估。
- **T5 tokenizer 缺失引发的训练失败给出可读错误（#387）**
  T5 tokenizer 本地目录缺失时训练会自动联网下载，此前网络不稳会报一段看不出原因的超时堆栈；现在下载开始前有日志提示，下载失败的错误信息写明原因与修复指引（检查网络 / 去设置页下载 / 核对路径配置）。
- **筛选页数据集切换对齐队列页样式（#392）**
  训练集 / 验证集切换从双 tab 改为与队列页一致的单按钮（显示要切去的数据集），移到页头最右。

### 修复

- **切换版本后的旧版本残留（#389, #390）**
  多版本项目切换版本后：立即点「开始训练」此前可能把旧版本入队；步骤页正文此前会残留旧版本数据——尤其图片筛选页会一直显示旧版本的图，看着旧图操作实际写入新版本。现在切换即时生效、步骤页整体随版本刷新；打标页有未保存编辑时切换会先弹确认。
- **终端 Ctrl+C 关不掉 server（#398）**
  浏览器 tab 开着时在终端按 Ctrl+C 停 server，此前会卡在「Waiting for connections to close」永远退不出（再按 Ctrl+C 也无效，只能杀进程）；现在一次 Ctrl+C 约 3 秒内干净退出，且不再刷出退出期的假报错。
- **Windows 训练日志不再出现 triton 报错误导（#388）**
  Windows 上启用 xformers 时，日志会打出无害的 `No module named 'triton'` 报错堆栈；任务因其他原因失败时它还会被误当失败原因展示，让人以为要装 triton。现在跳过该探测（triton 对本项目用到的 xformers 加速零参与），日志与失败摘要恢复干净。
