title: 访问地址改为根路径
version: "0.16.0"
---
Studio 的访问地址从 `http://<host>:<port>/studio/` 改为根路径 `http://<host>:<port>/`。

- 通过 studio.bat / `studio run` 启动会自动打开新地址，多数人无感。
- 旧的 `/studio/...` 书签仍可用：会自动跳转到新地址，建议把书签更新为根地址。
- 反向代理部署（如 HuggingFace Space、云 GPU 租赁机把容器端口代理到根路径）现在能正常打开了。

如果更新后页面空白，多半是浏览器缓存了旧页面：硬刷新（Ctrl+Shift+R）或用无痕窗口打开根地址即可。
