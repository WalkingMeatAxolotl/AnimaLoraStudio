NaViT 打包训练 · WandB 全局预设化 · 训练配置只落盘生效参数 · 去重审核精简 · CAME 优化器

### 新增

- **NaViT / Patch-n-Pack 打包训练（默认关闭）（#371, #372）**
  训练新增 `navit_packing` 模式：把多张不同尺寸的图打包进同一条序列训练，每张图只关注自己的内容，显存利用率由 token 预算决定，不再受「一批一个桶尺寸」限制。

  - 可选 `navit_native_resolution` 原生分辨率：不再把图缩放 / 裁剪到分桶网格，按原生尺寸参训（16px 对齐）；超出 token 预算时默认等比缩小，也可选直接报错。
  - 两者默认关闭；关闭时训练行为与之前完全一致。
- **timestep shift 分辨率修正（默认关闭）（#383）**
  训练配置新增 `timestep_shift_resolution_aware` 开关：开启后 timestep shift 按每张图的实际 token 数相对基准分辨率自动缩放（分辨率越高偏移越强），多分辨率训练与 NaViT 原生分辨率下每张图各自落在合适的噪声区段；与基准分辨率相同的图不受影响。
- **CAME 优化器（#358）**
  优化器新增 CAME：用行 / 列分解的二阶矩降低优化器状态的显存占用，并以置信度加权抑制分解带来的近似噪声；高级参数 6 项（beta / eps / clip），默认按论文取值。
- **LLM 打标可注入本地 tagger 预打标结果（#230）**
  LLM 预设新增「辅助 tagger」选项（关闭 / WD14 / CLTagger）：打标前先用本地 tagger 对每张图预打标，结果替换提示词中的 `{{tags}}` 占位符，作为先验辅助 LLM 写 caption。

  - 出厂自带两个含 `{{tags}}` 的内置辅助提示词预设（结构化 caption / 单行自然语言），也可在任意预设的提示词里手动加 `{{tags}}`；仅当提示词包含 `{{tags}}` 时生效。
- **训练配置页 / 预设页新增「YAML 预览」（#366）**
  两页可随时查看实际落盘、参训的生效配置；预设页原「TOML 预览」名不副实（字段并非 sd-scripts 格式），已由 YAML 预览取代。

### 变更

- **WandB 配置移入全局设置，训练配置中的 WandB 字段移除（#367）**
  WandB 从训练配置里的 per-config 覆盖块改为全局设置中的预设管理（设置 → WandB）：API Key 不再随 config.yaml / 任务快照 / 导出预设 / bundle 明文落盘。全局旧配置自动迁移；曾在训练配置里填过 WandB 的用户请看置顶迁移公告（含 API Key 轮换建议）。

  - 预设支持多套配置切换、另存为、导入导出（导出文件含真实 key，注意保管）。
  - artifact 上传设置从「采样图上传」的嵌套里独立出来，并合并为三档：关闭 / 仅保留最新 / 保留全部。
  - 命令行直跑训练的 `--wandb_*` 旗标随之移除，改用 `WANDB_*` 环境变量。
- **训练配置只落盘生效参数（#366, #369）**
  config.yaml / 预设 / 任务快照不再写入未启用功能的子参数与内部默认值，文件明显变短、查看参数没有噪音；已有文件读取不受影响，缺省键按默认值补齐。

  - 顺带修复：未启用功能的隐藏字段若残留非默认值（如切换优化器后留下的旧子参数），此前会静默参与训练；现在落盘即裁剪，所见即所训。
- **去重审核精简为两项扫描参数，裁剪并入分组（#381）**
  去重扫描参数从 18 项精简为「匹配范围 + 灵敏度（宽松 / 标准 / 严格）」两项，其余阈值固化为内置常量；裁剪 / 缩放关系不再是单独的「质量候选」tab，作为更严格的重复并进普通分组审核，且仅在「重复 + 同分镜差分」范围下检测。
- **打标页内直接管理模型（#378, #380）**
  打标页右栏改为「当前打标状态」面板（已打标数 / 打标方式 / 模型 / 触发词 / 此轮需要打标数）；模型下载、切换与下载源选择移入各打标器的「高级参数」折叠卡，不用再跳全局设置。

### 改进

- **侧边栏版本选择器重构为版本行（#384）**
  侧边栏「训练版本」块改为紧凑版本行：切换版本改为弹出浮层（不再撑开侧栏），hover 显示新建 / 导出 / 删除；总览页版本条同步补上删除入口，版本四操作两处对齐。
- **正则集页布局重构（#379）**
  「生成」tab 改为左右分栏：左侧生成参数按「正则集设置 / 出图设置」重新分组，右侧新增正则集状态面板，底部「此轮需要生成」按来源 / 模式 / 目标数实时推算，与后端口径一致。
- **预处理各页操作统一（#381）**
  工具切换改为贴 header 的下划线 tab，各页主操作统一移到页头右上角；裁剪页删掉左侧参数区给工作区让空间，「智能聚类」更名「自动聚类」并移入页头按钮弹层。
- **模型根目录迁移支持合并到已有数据（#368）**
  设置页迁移模型根目录时，目标已有数据不再直接拒绝：可选「跳过已有文件」（只补缺失文件，适合指向与其他工具共用的模型目录）或「覆盖已有文件」；复制全程逐文件原子写入，中途失败不会损坏目标目录的既有文件。
- **队列行操作按钮归位（#370）**
  「继续训练」与暂停行的「恢复 / 取消」从时间列移入行尾操作列并 icon 化；继续训练补上确认弹窗，防止误点直接拉起训练。
- **提示词选择器的数据集预览可点击放大（#375）**
  测试页「从训练集选 prompt」底部的图片预览支持点击放大全屏查看，方便与生成结果比对。

### 修复

- **出图中间步预览颜色反相（#377）**
  生成过程的中间步预览此前用错了解码器（按 Flux 而非 Wan2.1 的 latent 空间解码），颜色反相 / 错乱且画面很小；现已用正确的投影解码，预览颜色正常并铺满结果区。
- **回看历史时点「开始生成」结果区不再停在老图（#376）**
  在回看历史图（尤其 XY 图）时点「开始生成」，结果区此前停留在回看的老图上；现在会回到实时视图，直接看到新任务的生成进度。
- **只填一个代理字段时 HTTPS 流量不再绕过代理（#374）**
  HTTP / HTTPS 代理任一留空时回退用另一个（设置页一直如此描述，但此前未实现）：只填 HTTP 代理时 booru 查询 / 下载不再直连超时。两个字段都填时各自生效、互不覆盖。
- **方图分桶不再落到偏小的桶（#313）**
  多个候选桶宽高比距离相同时（如基准 1536 下的多个 1:1 桶），现在选面积最接近基准分辨率的桶，此前会落到第一个生成的偏小桶（1472×1472 而非 1536×1536）；前端桶预览与训练端同步该规则。受影响的数据集升级后分桶结果会略有变化。

### 删除

- **去重审核的「模糊候选」移除（#381）**
  模糊检测属于画质过滤、与「去重」无关，该 tab 及对应扫描逻辑已移除。
